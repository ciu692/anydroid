## 它到底能干什么？

说实话，大部分人第一次听到这类软件，脑子里浮现的就是"传个照片用的"。但 AnyDroid 能干的事情远不止这些。

**日常数据管理**：把手机连上电脑，联系人、短信、日历、应用……全都在电脑上分类呈现，看着比手机屏幕还清楚。想删哪个就删哪个，批量操作也完全没问题。

**跨平台文件传输**：手机和电脑互传文件，支持有线连接，也支持 WiFi 无线传输——不想找数据线的时候，扫个二维码就开始传，非常省事。

**数据备份与还原**：重要数据备份到电脑，万一手机出了意外也不慌。还支持从 iCloud 或 iTunes 导入数据到安卓设备，换机场景下特别好用。

**iOS 到安卓的数据迁移**：这个功能算是 AnyDroid 的亮点之一。从 iPhone 换到安卓？通讯录、照片、音乐、甚至部分消息记录都可以一键导过来，系统格式不兼容的内容会自动转换。

**应用批量管理**：把 APK 文件拖进软件就能直接安装到手机，也可以把手机上已经安装的 App 导出备份成 APK。

## 不用数据线也能传？

对，这是很多人试过之后觉得"这也太方便了"的功能。

AnyDroid 支持 WiFi 局域网连接，手机和电脑在同一个网络下，打开软件扫个码，连接就建立了。传照片、视频、音乐，跟插了线没什么差别，速度也不慢。

出门在外，手机里有视频想放到电脑上看？不用翻包找数据线，连个 WiFi 就搞定了。
